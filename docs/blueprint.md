# **App Name**: Monolith

## Core Features:

- Password Protected Access: Secure the entire application with a single, global password.
- Time Tracking: Track time spent on different projects.
- Task Management: Create tasks, linked to projects, with priorities, status and due dates
- Kanban Lists: Move tasks between sections freely.
- Habit Tracking: Track custom habits on weekly basis with differing frequencies.
- Dashboard overview: Most important information and quick navigation accessible for each module.
- Comprehensive Reporting: Create charts and generate visual reports based on time tracking and habit tracking data stored in Firestore.

## Style Guidelines:

- Dark background: Dark grey (#222222) for a professional, comfortable look.
- Primary color: Dusty rose (#C08497), for both subtlety and a professional appearance.
- Accent color: Soft gold (#D4AC0D) for subtle highlights and calls to action.
- Font pairing: 'Space Grotesk' for headings, 'Inter' for body text. Space Grotesk is a proportional sans-serif and Inter is a grotesque-style sans-serif.
- Use clean, minimalist icons for navigation and module representation.
- Dashboard layout: Clear grid-based layout with well-defined modules.
- Subtle transitions and animations on user interactions.
